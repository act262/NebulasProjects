# Nebulas Galley 

##  基于**Nebulas**区块链链上存储数据实现相册功能
### 使用
首先解锁用户钱包，
然后选择图片上传。

### 智能合约提供函数
`add` 上传图片

`get` 根据指定图片ID获取图片信息

`getAll` 获取当前保存的所有图片信息

`delete` 删除指定图片
